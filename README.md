## What is Data
raw unproccessed figures,facts,symbols or observations that  represent details about objects,events or phenomena examples of data include structured and unstructured data
## Explain the diffrence between structured and unstructured data
**structured data** This is a type of data that perfectly fits neatly to data tables such as numbers words and dates hence a have a fixed schema
**unstructured data** This is a data type that cannot perfectly fit in a data table hence doesnt have a fixed schema for example audio and images or even videos
## What is a schema
a schema is a structure or design of a database or database objects such as table,view,index,stored procedure for example we have a schema for what a door is and how it works
